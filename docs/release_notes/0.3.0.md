0.3.0
=====

- parameter extra_context is renamed to context and is now required in:
    - `render_front()` method in `Content` model
    - `render_admin()` method in `Content` model

- removed `show_front()` method in `Content`.  Override render_front() instead.
- removed `page` attribute in `Content` model. Use `section.container` instead.
- removed `create_for_page()` method in `Content` model. Use standard `{Model}.objects.create()`, position is set by frontend.
- removed `xprez_add_content_before_*` methods in `XprezAdminMixin`. Now handled by `xprez_add_content_*` methods.
